# KANNON BOSS MUSIC

A premium, high-performance distributed network landing page.

## Features

- **Atmospheric Visuals**: Custom cloud and sunshine ray animations.
- **Protocol Suite**: Detailed view of operational frameworks.
- **Global Network**: Live-sync node distribution status.
- **Premium Design**: Dark mode aesthetic with glassmorphism and high-quality typography.

## Tech Stack

- React + Vite
- Tailwind CSS
- Lucide React (Icons)
- Custom CSS Animations

## Deployment

Connected to Vercel for automated production deployments.
